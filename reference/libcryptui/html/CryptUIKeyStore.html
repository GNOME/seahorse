Property Details

The "keyset" property

  "keyset"                   CryptUIKeyset*        : Read / Write / Construct Only

Current CryptUI Key Source to use.

The "mode" property

  "mode"                     guint                 : Read / Write

Key store mode controls which keys to display.

Allowed values: <= 2

Default value: 0

The "none-option" property

  "none-option"              gchar*                : Read / Write

Text for row that denotes 'No Selection'.

Default value: ""

The "search" property

  "search"                   gchar*                : Read / Write

Key store search text for when in results mode.

Default value: ""

The "sortable" property

  "sortable"                 gboolean              : Read / Write

Allow user to sort columns.

Default value: FALSE

The "use-checks" property

  "use-checks"               gboolean              : Read / Write

Use check box column to denote selection.

Default value: FALSE
